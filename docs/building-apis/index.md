# Building APIs

This section covers all the features available in Pylight for building powerful APIs. Each feature is documented with examples and best practices.

## Available Features

### Core Features

- **[REST Endpoints](rest-endpoints.md)** - Automatic CRUD endpoint generation
- **[GraphQL](graphql.md)** - GraphQL queries and mutations
- **[WebSocket](websocket.md)** - Real-time WebSocket connections

### Security & Access Control

- **[Authentication](authentication.md)** - JWT and OAuth2 authentication with role-based access control

### Performance & Data Management

- **[Caching](caching.md)** - Redis caching for improved performance
- **[Pagination, Filtering, and Sorting](pagination-filtering-sorting.md)** - Built-in data management features

### Configuration & Setup

- **[YAML Configuration](yaml-configuration.md)** - Configure APIs using YAML files
- **[Database Reflection](database-reflection.md)** - Generate models from existing databases
- **[Plugins](plugins.md)** - Extend Pylight with custom plugins

## Quick Navigation

Choose a feature to learn more:

- Need to create REST APIs? Start with [REST Endpoints](rest-endpoints.md)
- Want GraphQL support? See [GraphQL](graphql.md)
- Need real-time updates? Check [WebSocket](websocket.md)
- Securing your API? Read [Authentication](authentication.md)
- Improving performance? Learn about [Caching](caching.md)
- Configuring via YAML? See [YAML Configuration](yaml-configuration.md)

## Feature Combinations

Many features work together:

- **REST + Authentication**: Secure REST endpoints with JWT/OAuth2
- **GraphQL + Caching**: Cache GraphQL query results
- **WebSocket + Authentication**: Secure WebSocket connections
- **YAML + All Features**: Configure everything via YAML
